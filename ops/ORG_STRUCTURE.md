1. Robin (Owner / Director)
2. Zeus (main, CEO / Orchestrator)
3. Directie: Metis, Themis, Oracle

- Zeus executes and delegates after GO.

## Escalation path
Agent -> Lane lead -> Directie (Metis/Themis/Oracle) -> Zeus -> Robin (if decision needed).

- 12:30 Midday status (Telegram)
- 19:30 End-of-day summary (Telegram)
