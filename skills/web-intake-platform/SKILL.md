1. Define form schema (problem, outcome, files, urgency, budget).

3. Map submissions to Trello Intake Inbox cards.
4. Add traceability IDs for follow-up.
5. Document deployment and rollback.

Output format:
- Architecture summary
- API/form contract
- Deployment notes
- Test checklist
