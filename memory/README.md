# Memory Log Protocol

Bestanden in deze map zijn append-only daglogs: `YYYY-MM-DD.md`.

## Entry-template

```md
## YYYY-MM-DD HH:mm Europe/Amsterdam — <agent>
Facts:
- ...

Decisions:
- ...

Ideas:
- ...

Artifacts:
- ...

Open actions:
- [owner] ... (deadline: ...)
```

## Regels
- Schrijf kort, feitelijk, verifieerbaar.
- Voeg altijd links/paden toe bij artifacts.
- Leg ideeën vast met hergebruik-context.
